Color Chart

CSS Color Chart is an outpouring of my passion for CSS Colors.

When looking for charts that had the comprehensive list, I was disappointed in how they were displayed. So I made my own and put all the colors in rainbow order. Because those are my favorite things.

For this project I used Github pages to host a simple HTML/CSS/Javascript site. I got some help with the Javascript for converting the namespace colors into hex values into RGB and in the process I learned a lot about how the two are related mathematically.

I enjoyed making this chart so much that I wanted to learn more about how they were named. I gave a talk at CSS Conf Oakland about what I found.
